# 파트론 SDK v1.0.0
## 사용조건
파트론 Temperature SDK는 누구나 사용할수 있습니다. SDK실행시 생기는 팝업을 임의로 훼손하거나  
변경하는 행위시 라이센스 위반으로 간주되며 확인 버튼을 선택시 SDK를 사용하겠다는 동의로 간주됩니다.  
  * 본 SDK는 PTD시리즈의 온도값외에는 다른 목적을 가지고 있지 않고 있습니다.  
  * 본 라인센스는 유효기간을 포함하고 있습니다.
